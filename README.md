勾选框说明：

0.1.7加入了勾选框。0.2.0后使用网页自带勾选框。

当勾选框全空时点击复制：

若搜索框内有关键词，自动勾选当前关键词匹配成功（如无关键词则为该字幕组全部）的磁链，搜索有效。

若搜索框内无关键词，则复制此字幕组下的所有磁链，搜索无效。

当勾选框有勾选时，复制勾选的磁链，搜索框及搜索模式无效。

关键字词搜索说明：

在输入框中输入的关键字词，与番组名进行严格匹配。

输入完成后点击“复制/打开”按钮即可。

留空则默认为不使用匹配功能。

匹配模式说明：

多词模式（默认）：以空格分割不同关键字词，复制所有关键字词都成功匹配的磁链。

排除模式：支持多词，排除所有番组名中含有任意一个关键词的磁链

单词模式：只能输入单个关键字词，输入的空格会当作关键字词进行匹配。

打开模式：便于一键打开磁链，自带多词搜索，也支持勾选框。

功能开关说明：

点击在字幕组列表文字后的齿轮图标以开关设置列表。

注：在关闭搜索框功能后，搜索模式以及保存/填充字幕组设置将同时被禁用。

自动展开说明：

会自动点击页面上的所有展开按钮一次，便于一次复制剧集或单集版本较少的番剧而不用手动点击展开。

设置存储/填充说明：

在你点击复制后，脚本将自动保存此次复制使用的模式和关键词信息在你本地，并将其与字幕组名关联。

下次打开番剧详情页时将会为此字幕组自动填充这些设置。

保存和填充设置功能可以在设置内开启/关闭。
